## Getting Started

### Prerequisites

- Node.js 18 or later
- npm or pnpm

### Installation

1. Clone the repository:

```bash
git clone <your-repo-url>
cd astro-app
```

2. Install dependencies:

```bash
npm install
# or
pnpm install
```

3. Start the development server:

```bash
npm run dev
# or
pnpm dev
```

The app will be available at `http://localhost:4321`
